# 一、最常用方式（页面里直接上传）⭐

## 操作步骤

1️⃣ 打开或编辑一个 Wiki 页面  
2️⃣ 将光标放在正文内容中  
3️⃣ 点击编辑器工具栏中的 **➕ / 📎 / Insert Asset**  
4️⃣ 选择 **Upload**  
5️⃣ 选择本地文件（PDF / Word / 图片 / ZIP 等）

📌 **上传完成后：**

- 文件会出现在 **Assets（资源库）** 中  
- 页面中会自动插入：
  - 文件下载链接  
  - 或图片 / 文件预览（取决于文件类型）

---

# 二、直接拖拽上传（最快）

## 操作方式

- 打开页面的编辑模式  
- 将本地文件 **直接拖拽到编辑器区域**

📌 **结果：**

- 文件会自动上传到 Assets  
- 页面中会自动插入对应内容：
  - 图片：直接显示  
  - 其他文件：插入下载 / 预览链接  

---

# 三、不同文件类型的页面表现

| 文件类型 | 页面表现 |
|--------|--------|
| 图片（png / jpg） | 直接显示 |
| PDF | 点击后在浏览器中预览 |
| Word / Excel | 下载（部分浏览器支持在线预览） |
| ZIP / TAR | 直接下载 |
| Markdown（.md） | 作为附件下载（不会自动渲染） |
